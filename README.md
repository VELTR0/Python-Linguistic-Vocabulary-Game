# Python-Linguistic-Vocabulary-Game
A python game for learning vocabularies for a linguistic course

## Description
An interactive vocabulary learning game built with Python and Pygame. Test your knowledge of vocabulary translations through a fun multiple-choice quiz format!

## Features
- Multiple choice quiz format
- Score tracking
- Real-time feedback on answers
- English to Spanish vocabulary (easily customizable)
- Simple and intuitive UI

## Requirements
- Python 3.6+
- Pygame 2.5.0+

## Installation

1. Clone the repository:
```bash
git clone https://github.com/VELTR0/Python-Linguistic-Vocabulary-Game.git
cd Python-Linguistic-Vocabulary-Game
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the game:
```bash
python game.py
```

## How to Play
1. Click "START" to begin the game
2. Read the English word displayed
3. Click on the correct translation from the multiple choice options
4. Your score will be tracked throughout the game
5. See your final score and accuracy at the end

## Customization
You can customize the vocabulary by editing the `vocabulary.py` file. Simply add or modify the word pairs in the dictionary format:
```python
vocabulary = {
    "english_word": "translation",
    ...
}
```
